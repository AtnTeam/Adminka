Відправка Push Notification для всіх підписників

Опис: Заголовок повідомлення.
Приклад значення: "Нове повідомлення"
Використання: Це текст, який буде відображатися в заголовку повідомлення.

Опис: Текст повідомлення.
Приклад значення: У вас нове повідомлення від користувача Іван."
Використання: Основне повідомлення, яке буде відображатися користувачу.

Опис:URL іконки повідомлення.
Приклад значення:"https://example.com/icon.png"
Використання: URL зображення, яке буде відображатися як іконка повідомлення. Якщо не вказано, використовується іконка за замовчуванням.

Опис:URL значка повідомлення (зазвичай використовується для відображення маленького значка в системному треї або на іконці додатку).
Приклад значення:"https://example.com/badge.png"
Використання:URL зображення, яке буде використовуватися як значок повідомлення. Зазвичай це маленьке зображення, яке відображається поверх основної іконки.

Опис:URL-адреса, яка буде відкрита при кліку на повідомлення.
Приклад значення:"https://example.com/messages"
Використання:Адреса, на яку буде перенаправлено користувача при кліку на повідомлення. Це корисно для направлення користувачів на конкретні сторінки вашого додатку або веб-сайту.
